Instructions to Run the Code:
------------------------------------------
1. Requirements:
   - Python 3.8+
   - scikit-learn
   - matplotlib
   - numpy
   - pandas
   - jupyter

   Install dependencies:
   pip install -r requirements.txt


2. Running the Notebook:
Open the Jupyter notebook:

Follow the notebook cells sequentially to preprocess data, train models (SVM, Random Forest, Decision Tree), and view evaluation metrics.

3. File Structure:
- `Project_Phase1.ipynb`: Main notebook with code and visualizations.
- `Test` and `Training`: Folder containing preprocessed image features and labels.
- `README.txt`: Instructions and GitHub link.
- `Project_report (AI_Bots).pdf`: Phase I report.

4. Dataset:
Dataset used is from MIT Places2:
http://places2.csail.mit.edu/download.html
(Subset used includes: Museum, Library, Shopping Mall – 5000 images/class)

5. Output:
The notebook generates metrics including accuracy, precision, recall, F1-score, and confusion matrices for all models.
